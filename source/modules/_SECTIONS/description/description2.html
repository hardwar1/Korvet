  • Оперативная доставка листа дюралевого Д16АТ по Туле и Тульской области

    Дюралевый лист — плоский прокат из сплавов алюминия. Он изготавливается согласно ГОСТ 21631-76. Химический состав соответствует ГОСТ 4784-74.

    Основные преимущества:

    • низкий вес;
    • высокая прочность;
    • пластичность;
    • устойчивость к коррозии;
    • огнестойкость.

    Дюралевый лист хорошо обрабатывается и неплохо поддается свариваемости.

    Он широко используется для создания прочных внешних конструкций, таких, как обшивки автомобилей и летательных аппаратов, кораблей. Рифленый лист применяется для создания противоскользящих, декоративных и облицовочных покрытий.
